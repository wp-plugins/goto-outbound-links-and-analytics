== Description ==
This plugin is designed for affiliates who want to hide the outbound links.

Analyze your clicks with graphs.

Hide your links.

Create and analyze goto links such as http://www.yoursite.com/goto/[link]/.

Ex:

- goto-url: http://www.google.com

- goto-link: google

- Your link: http://www.yoursite.com/goto/google/

- Redirect you to: http://www.google.com

== Installation ==

1. Upload directory `outbound-link-stats` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Visit Plugins menu to setup.
4. Fill in database information and goto-settings.
5. DONE!

== Frequently Asked Questions ==

= How do I add a link to track? =

Fill in URL, Goto, Name and select a parent if you want the link to be in a group.

= How do I select a parent? =

Select a radio button in the list of links.

= Can links have the same names? =

No! If you fill in a name that already exists, the link will automaticly assign itself to the name as a "child".
